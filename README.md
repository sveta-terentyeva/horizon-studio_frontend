# Horizon Studio Frontend Repository

## Build Setup
```
# install dependencies
$ npm install

# serve with hot reload for the development at localhost:3000
$ npm run dev

# build for production
$ npm run build

# serve build at localhost:8080
$ npm run start

Folder with build: 'dist/public'
```
